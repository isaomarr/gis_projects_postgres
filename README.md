# GIS Projects – PostgreSQL Demo

Bu kiçik layihə PostgreSQL üzərində GIS layihələri üçün sadə bir cədvəl nümunəsidir.  
Məqsəd: `schema.sql`, `data.sql` və `queries.sql` faylları ilə GitHub-da praktik SQL nümunəsi göstərməkdir.

## Fayllar

- `schema.sql` – `gis_projects` cədvəlinin strukturu (schema)
- `data.sql` – nümunə GIS layihə məlumatları (INSERT)
- `queries.sql` – müxtəlif SELECT sorğuları nümunələri
